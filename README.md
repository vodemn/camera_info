# camera_info

A Flutter plugin that exposes per-camera optical metadata on iOS and Android — information that the standard `camera` plugin does not surface.

## Platform support

| Android | iOS |
|:-------:|:---:|
| ✅      | ✅  |

## Installation

```yaml
dependencies:
  camera_info: ^0.0.1
```

## Usage

```dart
import 'package:camera_info/camera_info.dart';

final plugin = CameraCapabilities();
final cameras = await plugin.getCameraCapabilities();

for (final cam in cameras) {
  print('Focal length : ${cam.focalLength} mm');
  print('Aperture     : f/${cam.aperture}');
  print('Field of view: ${cam.fieldOfView}°');
  print('Zoom range   : ${cam.minZoomFactor}x – ${cam.maxZoomFactor}x');
  print('EV step size : ${cam.exposureOffsetStepSize} EV');
}
```

## Data fields

| Field | Type | Description |
|---|---|---|
| `focalLength` | `double?` | Focal length in millimetres |
| `aperture` | `double?` | Maximum aperture as an f-number (e.g. `1.8` → f/1.8) |
| `fieldOfView` | `double?` | Horizontal field of view in degrees |
| `minZoomFactor` | `double?` | Minimum supported zoom factor |
| `maxZoomFactor` | `double?` | Maximum supported zoom factor |
| `exposureOffsetStepSize` | `double?` | Smallest EV step for exposure compensation |

All fields are nullable — not every platform exposes every value for every camera.

## Platform notes

**iOS** — uses AVFoundation (`AVCaptureDevice`): `lensAperture`, `videoFieldOfView`, `minAvailableVideoZoomFactor`, `maxAvailableVideoZoomFactor`.

**Android** — uses the Camera2 API (`CameraCharacteristics`): `LENS_INFO_AVAILABLE_FOCAL_LENGTHS`, `LENS_INFO_AVAILABLE_APERTURES`, `SCALER_AVAILABLE_MAX_DIGITAL_ZOOM`, `CONTROL_AE_COMPENSATION_STEP`.

## Development

Platform channels are generated by [Pigeon](https://pub.dev/packages/pigeon). Do not hand-write method channel code.

**Pigeon input:** `pigeons/camera_info.dart`

**Generated files** (committed to git — consumers only need `pub get`):
- `lib/src/camera_info.g.dart`
- `android/src/main/kotlin/com/vodemn/camera_info/CameraCapabilitiesApi.g.kt`
- `ios/Classes/CameraCapabilitiesApi.g.swift`

To regenerate after editing the pigeons file:

```
flutter pub run pigeon --input pigeons/camera_info.dart
```
